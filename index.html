Project report

Cursor Glove

A wireless glove consisting of a single IMU, which tracks a 3D Visualization of estimated position & orientation of the hand movements using filters. The main application is to completely replace the mouse and use the glove to control the mouse pointer on any PC. This glove is used as a peripheral device for other applications like gaming, Virtual Reality and Augmented Reality.

UPPSALA UNIVERSITY 2017-2020

M.Sc. in Embedded systems
The thesis is to create a prototype of a wearable device for intuitive interaction with the 3D 'surface'. A glove that can identify different hand gestures with the help of IMU sensors and data from the sensors is estimated using different filtration techniques(built on Matlab) and this data runs through a machine-learning algorithm to identify different hand gestures. The applications we target are Sign language recognition and interaction with an Augmented reality environment as a proof of concept that a single glove can be used for multiple applications.
