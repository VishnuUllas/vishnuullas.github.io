Projects

Master’s Thesis

The design and development of a smart glove embedded with inertial measurement units, used to recognize Hand Activity/Gestures using Machine learning algorithms and Neural Networks. This prototype is used to integrate with multiple applications based on 3D interactions. The main applications the prototype targets are Sign language recognition and interaction with an Augmented reality environment.

Project report

Cursor Glove

A wireless glove consisting of a single IMU, which tracks a 3D Visualization of estimated position & orientation of the hand movements using filters. The main application is to completely replace the mouse and use the glove to control the mouse pointer on any PC. This glove is used as a peripheral device for other applications like gaming, Virtual Reality and Augmented Reality.

Project report

Elevator Model

A software model of an elevator built using MATLAB and Simulink. This software is used to simulate a miniature model of an elevator which is built for testing various Simulink models that are code generated.

Sweet Shot

RFID based speed monitoring system

M.Sc. in Embedded systems
The thesis is to create a prototype of a wearable device for intuitive interaction with the 3D 'surface'. A glove that can identify different hand gestures with the help of IMU sensors and data from the sensors is estimated using different filtration techniques(built on Matlab) and this data runs through a machine-learning algorithm to identify different hand gestures.
